
#  *Asteroids*

In this project, you will recreate the classic arcade game **Asteroids** using **Pygame**.

You will program:

* A rotating spaceship
* Asteroids that drift across space
* Bullets that destroy asteroids
* Screen wrap mechanics
* Collision detection
